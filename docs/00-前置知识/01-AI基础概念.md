# 一、AI 基础概念

在开始学习 AI Agent 之前，我们需要先掌握一些核心的基础概念。本章将介绍人工智能、机器学习、深度学习等领域的基本概念和它们之间的关系。

## 1. 人工智能

**人工智能（Artificial Intelligence，简称 AI）** 是计算机科学的一个分支，致力于创建能够执行通常需要人类智能才能完成的任务的系统。这些任务包括视觉感知、语音识别、决策制定、语言翻译等。

### 1.1 AI 的发展历程

人工智能的发展可以分为几个阶段：

1. **早期探索（1950s-1970s）**：AI 概念的提出，早期专家系统的发展
2. **AI 寒冬（1970s-1980s）**：由于技术限制和期望过高，AI 研究进入低谷
3. **专家系统复兴（1980s-1990s）**：基于规则的专家系统得到广泛应用
4. **机器学习时代（1990s-2010s）**：统计学习方法兴起
5. **深度学习革命（2010s-至今）**：神经网络和深度学习技术取得重大突破

### 1.2 AI 的分类

AI 可以按照能力水平分为：

| 类型 | 描述 | 示例 |
|------|------|------|
| 狭义 AI（Weak AI） | 专注于完成特定任务的 AI 系统 | 语音助手、推荐系统、图像识别 |
| 通用 AI（Strong AI） | 具备与人类相当的智能，能够完成任何智力任务 | （尚未实现） |
| 超级 AI（Super AI） | 智能水平超越人类的 AI | （理论概念） |

目前我们实际应用的 AI 系统都属于狭义 AI 范畴。

## 2. 机器学习

**机器学习（Machine Learning，简称 ML）** 是人工智能的一个子领域，它使计算机系统能够通过经验自动改进，而无需显式编程。

### 2.1 机器学习的核心思想

传统编程是：**数据 + 程序 = 结果**

机器学习是：**数据 + 结果 = 程序**

机器学习算法从大量数据中学习模式和规律，然后利用学到的知识进行预测或决策。

### 2.2 机器学习的主要类型

机器学习可以分为以下几类：

#### 2.2.1 监督学习（Supervised Learning）

监督学习使用带标签的训练数据进行学习。模型学习从输入到输出的映射关系。

**常见算法**：
- 线性回归（Linear Regression）
- 逻辑回归（Logistic Regression）
- 决策树（Decision Tree）
- 随机森林（Random Forest）
- 支持向量机（SVM）
- 神经网络（Neural Network）

**应用场景**：
- 房价预测
- 垃圾邮件分类
- 图像识别

#### 2.2.2 无监督学习（Unsupervised Learning）

无监督学习使用没有标签的数据进行学习，目标是发现数据中的隐藏结构或模式。

**常见算法**：
- 聚类（Clustering）：K-Means、层次聚类
- 降维（Dimensionality Reduction）：PCA、t-SNE
- 关联规则挖掘

**应用场景**：
- 客户分群
- 异常检测
- 数据可视化

#### 2.2.3 强化学习（Reinforcement Learning）

强化学习通过智能体与环境的交互来学习。智能体通过试错获得奖励信号，学习如何最大化长期奖励。

**常见算法**：
- Q-Learning
- DQN（Deep Q-Network）
- 策略梯度（Policy Gradient）
- PPO（Proximal Policy Optimization）

**应用场景**：
- 游戏 AI
- 机器人控制
- 推荐系统优化

## 3. 深度学习

**深度学习（Deep Learning）** 是机器学习的一个子领域，基于人工神经网络，特别是深层神经网络（具有多层结构的神经网络）。

### 3.1 为什么叫"深度"

深度学习中的"深度"指的是神经网络的层数。传统的神经网络可能只有 2-3 层，而深度神经网络可以有几十甚至上百层。

### 3.2 常见的深度学习架构

#### 3.2.1 卷积神经网络（Convolutional Neural Network，CNN）

**特点**：擅长处理网格状数据（如图像）

**核心组件**：
- 卷积层（Convolutional Layer）
- 池化层（Pooling Layer）
- 全连接层（Fully Connected Layer）

**应用**：图像识别、目标检测、图像分割

#### 3.2.2 循环神经网络（Recurrent Neural Network，RNN）

**特点**：擅长处理序列数据（如文本、时间序列）

**改进版本**：
- LSTM（Long Short-Term Memory）
- GRU（Gated Recurrent Unit）

**应用**：语言模型、机器翻译、语音识别

#### 3.2.3 Transformer

**特点**：基于自注意力机制，能够并行处理序列数据

**核心组件**：
- 自注意力（Self-Attention）
- 多头注意力（Multi-Head Attention）
- 位置编码（Positional Encoding）

**应用**：大语言模型、机器翻译、文本生成

### 3.3 深度学习的优势

1. **自动特征学习**：无需手动设计特征，模型可以自动学习有用的特征表示
2. **性能优异**：在许多任务上超越传统方法
3. **可扩展性**：随着数据量和计算资源的增加，性能可以持续提升

## 4. 神经网络

**神经网络（Neural Network）** 是一种模仿人脑神经元结构的计算模型，是深度学习的基础。

### 4.1 神经元模型

单个神经元接收输入，进行加权求和，然后通过激活函数产生输出：

```
输出 = 激活函数(权重1 × 输入1 + 权重2 × 输入2 + ... + 偏置)
```

### 4.2 常见的激活函数

| 激活函数 | 公式 | 特点 |
|---------|------|------|
| Sigmoid | σ(x) = 1/(1+e⁻ˣ) | 输出在 (0,1) 之间，可表示概率 |
| ReLU | ReLU(x) = max(0, x) | 计算简单，缓解梯度消失 |
| Tanh | tanh(x) | 输出在 (-1,1) 之间 |
| Softmax | - | 将输出转换为概率分布 |

### 4.3 神经网络的训练

神经网络训练通常包括以下步骤：

1. **前向传播**：输入通过网络，计算输出
2. **损失计算**：比较预测输出和真实输出，计算损失
3. **反向传播**：计算损失对各参数的梯度
4. **参数更新**：使用梯度下降等优化算法更新权重和偏置

## 5. 自然语言处理

**自然语言处理（Natural Language Processing，简称 NLP）** 是 AI 的一个重要分支，致力于让计算机理解、解释和生成人类语言。

### 5.1 NLP 的主要任务

| 任务类型 | 示例 |
|---------|------|
| 文本分类 | 情感分析、主题分类 |
| 序列标注 | 命名实体识别、词性标注 |
| 文本生成 | 文章写作、对话系统 |
| 信息抽取 | 关系抽取、事件抽取 |
| 机器翻译 | 多语言翻译 |
| 问答系统 | 智能问答、知识库问答 |

### 5.2 NLP 技术的发展

1. **规则时代**：基于人工编写的规则
2. **统计时代**：基于统计学习方法
3. **深度学习时代**：基于神经网络
4. **大模型时代**：基于预训练大语言模型

## 6. 概念之间的关系

AI、机器学习、深度学习、神经网络之间的关系可以用以下层次表示：

```
人工智能（AI）
├── 机器学习（ML）
│   ├── 深度学习（DL）
│   │   └── 神经网络（Neural Network）
│   ├── 其他 ML 方法（SVM、决策树等）
├── 其他 AI 方法（专家系统、搜索算法等）
```

## 7. 总结

本章介绍了 AI 领域的核心基础概念：

- **人工智能（AI）**：创建智能系统的科学
- **机器学习（ML）**：通过数据学习的 AI 方法
- **深度学习（DL）**：基于深层神经网络的机器学习
- **神经网络**：模仿人脑的计算模型
- **自然语言处理（NLP）**：处理人类语言的 AI 技术

这些概念是后续学习 AI Agent 的重要基础。理解它们之间的关系和各自的特点，将有助于我们更好地掌握 AI Agent 技术。
