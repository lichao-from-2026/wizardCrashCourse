# 一、提示词模式汇总

汇总所有常用的提示词模式，帮助你在不同场景下选择合适的模式。

## 1. 模式对比

| 模式 | 特点 | 适用场景 | 效果 | 成本 |
|------|------|---------|------|------|
| Zero-shot | 简单直接 | 简单任务 | ⭐⭐ | 低 |
| Few-shot | 提供示例 | 需要示例引导 | ⭐⭐⭐ | 中 |
| Chain-of-Thought | 展示推理 | 复杂推理 | ⭐⭐⭐⭐ | 中高 |
| ReAct | 推理+行动 | 需要使用工具 | ⭐⭐⭐⭐⭐ | 高 |
| Self-Consistency | 多答案投票 | 高准确率要求 | ⭐⭐⭐⭐⭐ | 高 |

## 2. 如何选择模式

### 2.1 决策树

```
开始
  |
  +-- 任务简单？
  |     |
  |     +-- 是 → Zero-shot
  |     |
  |     +-- 否 → 需要示例？
  |            |
  |            +-- 是 → Few-shot
  |            |
  |            +-- 否 → 需要推理？
  |                   |
  |                   +-- 是 → 需要工具？
  |                   |     |
  |                   |     +-- 是 → ReAct
  |                   |     +-- 否 → CoT
  |                   |
  |                   +-- 否 → 需要高准确率？
  |                         |
  |                         +-- 是 → Self-Consistency
  |                         +-- 否 → Zero-shot 或 Few-shot
```

### 2.2 快速选择

| 需求 | 推荐模式 |
|------|---------|
| 快速回答 | Zero-shot |
| 格式要求 | Few-shot |
| 逻辑推理 | CoT |
| 使用工具 | ReAct |
| 高准确率 | Self-Consistency + CoT |

**示例：**
```
Few-shot + CoT + ReAct
```
