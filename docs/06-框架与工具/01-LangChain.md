# 一、LangChain

LangChain 是目前最流行的 LLM 应用开发框架！

## 1. 核心概念

| 概念 | 说明 |
|------|------|
| **LLM** | 大语言模型封装 |
| **Prompt Template** | 提示词模板 |
| **Chain** | 多个组件链接在一起 |
| **Agent** | 具有推理和工具调用能力 |
| **Memory** | 记忆系统 |
| **Retriever** | 检索器（RAG） |

## 2. 优势

- 生态丰富，组件多
- 社区活跃，文档全
- 支持多语言和多模型

## 3. 适用场景

- RAG 系统
- 聊天机器人
- 工作流自动化
- 多 Agent 协作

## 4. 总结

LangChain 是入门 Agent 开发的绝佳选择！
