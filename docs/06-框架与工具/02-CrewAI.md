# 一、CrewAI

CrewAI 是专门为多 Agent 协作设计的框架！

## 1. 核心概念

| 概念 | 说明 |
|------|------|
| **Agent** | 具有角色、目标、工具 |
| **Task** | 分配给 Agent 的任务 |
| **Crew** | 多个 Agent 组成的团队 |
| **Process** | 任务执行流程（顺序、并行） |

## 2. 特点

- 专注多 Agent 协作
- 角色定义清晰
- 任务编排灵活
- 支持工具共享

## 3. 适用场景

- 团队协作工作流
- 复杂任务分解
- 角色扮演应用

## 4. 总结

CrewAI 是多 Agent 场景的首选框架！
