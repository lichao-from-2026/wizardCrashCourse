# 一、AutoGen

AutoGen 是 Microsoft 开发的多 Agent 对话框架！

## 1. 核心概念

| 概念 | 说明 |
|------|------|
| **Agent** | 可对话的智能体 |
| **Conversation** | Agent 之间的对话 |
| **Tool Use** | 工具调用 |
| **Human-in-the-loop** | 人机交互 |

## 2. 特点

- 由 Microsoft 支持
- 强调 Agent 对话
- 支持代码执行
- 灵活的交互模式

## 3. 适用场景

- 人机协作
- 代码生成与执行
- 研究实验

## 4. 总结

AutoGen 适合研究和需要人机协作的场景！
