# 一、项目二：研究助手 Agent

这是一个帮助你做研究的助手！

## 1. 功能设计

- 文献检索
- 资料整理
- 内容总结
- 观点对比
- 引用生成

## 2. 技术要点

| 技术 | 用途 |
|------|------|
| **RAG** | 检索和整合外部知识 |
| **提示词工程** | 指导 LLM 做研究任务 |
| **记忆系统** | 记住研究过程 |

## 3. 实现思路

1. 构建知识库
2. 设计 RAG 检索
3. 编排研究流程
4. 优化输出质量

## 4. 总结

研究助手能大幅提升信息收集效率！
