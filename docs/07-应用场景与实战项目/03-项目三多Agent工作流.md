# 一、项目三：多 Agent 工作流

让多个 Agent 协作完成任务！

## 1. 场景设计

比如内容创作：
- **研究员 Agent** → 查资料
- **撰稿人 Agent** → 写初稿
- **编辑 Agent** → 审稿修改
- **发布者 Agent** → 格式化发布

## 2. 技术要点

| 技术 | 用途 |
|------|------|
| **多 Agent 协作** | Agent 之间沟通配合 |
| **任务分解** | 把大任务拆成小任务 |
| **管道编排** | 按顺序/并行执行 |

## 3. 实现思路

1. 定义每个 Agent 的角色
2. 设计任务流转
3. 处理 Agent 间的沟通
4. 测试和优化

## 4. 总结

多 Agent 能完成单个 Agent 做不到的复杂任务！
