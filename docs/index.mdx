# AI Agent 实践学习知识库

## 项目简介

本项目是一个基于 RsPress 构建的 AI Agent 知识库，旨在为学习者提供从零开始的 AI Agent 学习路径和系统化的知识整理方案。

## ✨ 三大核心工程

本知识库重点突出以下三大核心工程，这些是构建高质量 AI Agent 的关键：

| 工程 | 描述 | 学习重点 |
|------|------|---------|
| **提示词工程** | 设计和优化提示词，引导 LLM 更好地推理和输出 | Zero-shot、Few-shot、Chain-of-Thought、ReAct、提示词模式 |
| **上下文工程** | 有效管理和利用上下文窗口，构建高效的 RAG 系统 | 上下文窗口管理、检索优化、长文本处理、知识检索 |
| **Harness 工程** | 构建工程化、可维护、可观测的 Agent 系统 | 模块化设计、管道编排、状态管理、可观测性、测试评估 |

## 📚 学习路径

按照以下路径系统学习 AI Agent：

1. **前置知识** → 学习 AI 基础概念、LLM 基础、核心技术
2. **Agent 基础理论** → 理解 Agent 的定义、组件、架构
3. **提示词工程** → 掌握提示词设计和优化（核心重点）
4. **上下文工程** → 学习 RAG 和上下文管理（核心重点）
5. **Harness 工程** → 工程化实践（核心重点）
6. **技术栈其他组件** → 记忆、工具、规划等
7. **框架与工具** → LangChain、CrewAI、AutoGen 等
8. **实战项目** → 应用所学完成项目
9. **最佳实践** → 在项目中的实际应用指南
