# 项目配置说明

## package.json - 项目依赖和脚本配置

```json
{
  "name": "ai-agent-knowledge-base",        // 项目名称 - AI Agent 知识库
  "version": "1.0.0",                       // 版本号
  "private": true,                          // 是否为私有项目
  "description": "AI Agent 实践学习知识库", // 项目描述
  "scripts": {
    "dev": "rspress dev",                   // 开发模式 - 启动本地开发服务器
    "build": "rspress build",               // 构建命令 - 生成静态站点
    "preview": "rspress preview"            // 预览命令 - 预览构建后的站点
  },
  "dependencies": {
    "react": "^18.2.0",                    // React 框架 - 用户界面库
    "react-dom": "^18.2.0"                 // React DOM - DOM 操作库
  },
  "devDependencies": {
    "rspress": "^1.35.0",                  // RsPress - 静态站点生成器
    "typescript": "^5.0.0",                // TypeScript - 类型安全的 JavaScript
    "@types/react": "^18.2.0",            // React 类型定义
    "@types/react-dom": "^18.2.0"         // React DOM 类型定义
  }
}
```

## tsconfig.json - TypeScript 编译器配置

```json
{
  "compilerOptions": {
    "target": "ES2020",                              // 编译目标 - 输出为 ES2020 版本
    "lib": ["ES2020", "DOM", "DOM.Iterable"],       // 类型库 - 包含 ES2020 和 DOM 类型
    "jsx": "react-jsx",                              // JSX 转换 - 使用 React 17+ 新的 JSX 转换
    "module": "ESNext",                              // 模块系统 - 使用最新的 ES 模块
    "skipLibCheck": true,                            // 跳过库检查 - 提升编译速度
    "moduleResolution": "bundler",                   // 模块解析 - 使用 bundler 模式
    "allowImportingTsExtensions": true,              // 允许导入 TS 扩展名
    "resolveJsonModule": true,                        // 允许导入 JSON 模块
    "isolatedModules": true,                           // 隔离模块 - 确保文件可以独立编译
    "noEmit": true,                                   // 不输出文件 - 只进行类型检查
    "strict": true,                                   // 启用严格模式 - 严格的类型检查
    "noUnusedLocals": true,                           // 检查未使用的局部变量
    "noUnusedParameters": true,                       // 检查未使用的参数
    "noFallthroughCasesInSwitch": true,              // 检查 switch 语句中的穿透
    "esModuleInterop": true,                          // ES 模块互操作 - 兼容 CommonJS 模块
    "allowSyntheticDefaultImports": true,             // 允许合成默认导入
    "forceConsistentCasingInFileNames": true         // 强制文件名大小写一致
  },
  "include": ["rspress.config.ts"],                   // 包含的文件 - 只检查配置文件
  "exclude": ["node_modules"]                         // 排除的文件 - 不检查 node_modules
}
```
